# SPC Simulators

Interactive, self-contained SPC (Statistical Process Control) capability and control-chart simulators. Each is a single standalone HTML file — no build step, no dependencies.

**Live demos:** https://manojkumar-alagesan.github.io/spc-simulators/

- [Individuals (I-MR) capability simulator](https://manojkumar-alagesan.github.io/spc-simulators/spc-individuals-chart.html) — one measurement at a time. Live Cp/Cpk/Pp/Ppk against AIAG SPC manual bands, plus I-MR control limits from a moving-range baseline.
- [Subgroups (X̄–R / X̄–S) capability simulator](https://manojkumar-alagesan.github.io/spc-simulators/spc-subgroups-chart.html) — rational subgroups of n parts per sample. Short-term Cp/Cpk from R̄/d2 or s̄/c4, overall Pp/Ppk from pooled individuals, plus X̄ and R/S control charts.

## Running locally

Just open the `.html` file in a browser — everything (styles, scripts, math) is inlined in the file.
